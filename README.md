# 🧜‍♀️ MerMommy Creations - Landing Page

Welcome to the official repository for the **MerMommy Creations** splash landing page! This website features a high-energy, whimsical, 3D cartoon underwater theme built to showcase premium event curation, party themes, and luxury decor styling services.

---

## ✨ Features

- **Smooth Cartoon Animations:** Built using **GSAP (GreenSock Animation Platform)** to create a playful entrance bounce and infinite looping visual elements.
- **Dynamic Particle Spawner:** Custom JavaScript loops that continuously float responsive bubbles and magical sparkles (`✨`) from the ocean floor.
- **Vibrant Theme Matrix:** A responsive grid highlighting signature custom event curations (Lemonade Stand, In-N-Out Burger, Tie-Dye Studio, and the Mermaid Kingdom masterpiece) with custom 3D layered hover effects.
- **Fully Integrated Booking System:** A clean, accessible inquiry form pre-styled to drop right into booking processors like Formspree.
- **Responsive Layout:** Designed from the mobile screen up using modern CSS Grid and Flexbox mechanics to ensure it looks beautiful on devices of all sizes.
