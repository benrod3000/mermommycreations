# 🧜‍♀️ MerMommy Creations - Premium Party Curation

Welcome to the official repository for the **MerMommy Creations** landing page! This website features an immersive, ocean-inspired design with glass-morphism effects, showcasing premium event curation, party themes, and luxury decor styling services.

---

## ✨ Features

- **Ocean-Inspired Visual Design:** A stunning deep-sea color palette with oceanic blues and teals, replacing the original purple theme for a more cohesive water-inspired brand identity.
- **Glass-Morphism UI:** All cards feature a transparent, frosted glass effect with `backdrop-filter: blur(8px)` creating a modern, premium look throughout the entire site.
- **Smooth GSAP Animations:** Built using **GSAP (GreenSock Animation Platform)** for playful entrance bounces, staggered card reveals, and infinite floating icon loops.
- **Dynamic Particle System:** Custom JavaScript that continuously floats responsive bubbles and magical sparkles (`✨`) from the ocean floor, creating an immersive underwater atmosphere.
- **Enhanced Adventure Grid:** A responsive 4-column card grid showcasing signature party themes with:
  - Themed color accents
  - Feature chips highlighting key inclusions
  - Starting price badges for transparency
  - Consistent, balanced descriptions
  - Hover lift effects
- **Fully Integrated Booking System:** A clean, accessible inquiry form pre-styled with glass-morphism inputs and Formspree integration for seamless booking.
- **Responsive Layout:** Designed mobile-first using CSS Grid and Flexbox, ensuring beautiful display across all devices from mobile to desktop.
